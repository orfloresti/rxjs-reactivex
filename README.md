## To start development mode

`Run in different consoles`

Init buil and watch

```bash
npm run start:build
```

Init a nodemon

```bash
npm run start:dev
```

## Create a new project

Create new project

```bash
npm i typescript --save-dev
```

Init typescript config projects

```bash
npx tsc --init
```
